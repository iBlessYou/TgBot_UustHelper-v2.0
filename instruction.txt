Инструкция по запуску приложения на сервер

1. Создаем рабочую директорию
2. Создаем виртуальную среду    python -m venv venv.
3. Активируем виртуальную среду source venv/bin/activate (деактивировать - deactivate)
4. Создаем конфигурацию библиотек pip freeze > requirements.txt , распаковываем в рабочей директории pip install -r requirements.txt
4. Копируем архив
5. Запускаем исполняемые файлы
